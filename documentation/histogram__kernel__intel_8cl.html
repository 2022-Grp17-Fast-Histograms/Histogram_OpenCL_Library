Kernel function that calculates the histograms for all channels. The kernel calculates the average and variance histograms based on the average and variance of each group (block of pixels). It accepts YUV/NV12 format and performs the calculation for all channels (YUV).

Parameters
pixelspointer to raw image data.
numOfBinsthe number of bins on the calculated histograms.
formatthe format of the raw image data. 0 = YUV, 1 = NV12.
yAverageBinsthe histogram data for the average for channel Y.
yVarianceBinsthe histogram data for the variance for channel Y.
uAverageBinsthe histogram data for the average for channel U.
uVarianceBinsthe histogram data for the variance for channel U.
vAverageBinsthe histogram data for the average for channel V.
vVarianceBinsthe histogram data for the variance for channel V.
yBlockSumAveragelocal memory for the accumulative sum (reduction) for the the average of the group for channel Y.
yBlockSumVariancelocal memory for the accumulative sum (reduction) for the the variance of the group for channel Y.
uBlockSumAveragelocal memory for the accumulative sum (reduction) for the the average of the group for channel U.
uBlockSumVariancelocal memory for the accumulative sum (reduction) for the the variance of the group for channel U.
vBlockSumAveragelocal memory for the accumulative sum (reduction) for the the average of the group for channel V.
vBlockSumVariancelocal memory for the accumulative sum (reduction) for the the variance of the group for channel V.
